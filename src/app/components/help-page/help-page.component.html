Explication des Graphiques

Comprendre vos données d'inspection

Cette page vous aide à interpréter les différents graphiques présents dans votre tableau de bord. Chaque section explique un graphique spécifique et comment utiliser les informations qu'il présente.

Évolution des rapports d'inspection par partenaire

À quoi sert ce graphique ?

Ce graphique linéaire montre l'évolution du nombre de rapports d'inspection produits par chaque partenaire au fil du temps.

Comment l'interpréter ?

  • Chaque ligne colorée représente un partenaire différent
  • L'axe horizontal (X) indique les dates
  • L'axe vertical (Y) montre le nombre de rapports
  • Les pics indiquent des périodes d'activité intense
  • Les baisses peuvent révéler des périodes moins actives

Exemple d'analyse

Si vous observez une ligne qui monte régulièrement, cela indique que le partenaire concerné augmente son activité d'inspection. À l'inverse, une baisse soudaine pourrait nécessiter une investigation.

Actions possibles

  • Identifier les partenaires les plus actifs
  • Détecter les tendances saisonnières
  • Comparer la performance entre partenaires

Inspections conformes vs non conformes

À quoi sert ce graphique ?

Ce graphique à barres compare quotidiennement le nombre d'inspection conformes et non conformes.

Comment l'interpréter ?

  • Chaque paire de barres représente un jour
  • La barre "Conformes" montre les inspections valides
  • La barre "Non conformes" indique les problèmes détectés
  • La hauteur des barres représente le volume
  • Le ratio entre les deux donne le taux de conformité

Exemple d'analyse

Un jour où les barres non conformes sont particulièrement hautes par rapport aux conformes peut indiquer un problème spécifique qui mérite investigation.

Actions possibles

  • Surveiller le taux de conformité quotidien
  • Identifier les jours à problèmes
  • Analyser les causes des non-conformités

Répartition par type de véhicule

À quoi sert ce graphique ?

Ce graphique circulaire (camembert) montre la proportion des différents types de véhicules inspectés.

Comment l'interpréter ?

  • Chaque section représente un type de véhicule
  • La taille de la section indique sa proportion
  • Le pourcentage précise la part de chaque type
  • Les couleurs aident à différencier les catégories

Exemple d'analyse

Si les camions représentent 98.5% des inspections, cela signifie que votre activité est principalement concentrée sur ce type de véhicule.

Actions possibles

  • Adapter les ressources aux types de véhicules majoritaires
  • Identifier les niches ou marchés émergents
  • Planifier les formations spécifiques

Évolution des inspections

À quoi sert ce graphique ?

Ce graphique montre la tendance globale du volume d'inspections réalisées jour après jour.

Comment l'interpréter ?

  • La ligne représente l'évolution du nombre total d'inspections
  • Les points marquent les valeurs quotidiennes
  • La pente indique si l'activité augmente ou diminue
  • Les écarts entre points montrent la variabilité

Exemple d'analyse

Une tendance à la hausse régulière indique une croissance de l'activité. Des baisses soudaines peuvent révéler des problèmes opérationnels ou des jours fériés.

Actions possibles

  • Planifier les ressources en fonction des tendances
  • Anticiper les périodes de forte activité
  • Analyser l'impact des actions commerciales

Conseils pour une analyse efficace

Utilisez les filtres

N'oubliez pas d'utiliser les filtres par date et partenaire pour affiner votre analyse et obtenir des données plus précises.

Comparez les périodes

Pour une analyse pertinente, comparez toujours les données sur différentes périodes (semaine vs semaine précédente, mois vs même mois l'année dernière).
